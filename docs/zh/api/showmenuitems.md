# easi.showMenuItems

批量显示菜单项

sdk 成功返回值

无

示例

```TypeScript
easi.showMenuItems({
  menuItems: []
})
```

## 参数

| 参数      | 描述         | 类型     | 值                                                        | 是否必须 |
| --------- | ------------ | -------- | --------------------------------------------------------- | -------- |
| menuItems | 隐藏的菜单项 | string[] | ['wechat','moment','facebook','browser','refresh','copy'] | 是       |
